## Local SVG Icons Location

**Path**: `mobile-version/src/assets/icons/`

All 90 local SVG icons are stored in this folder. They are also available as React components via the index file:

```typescript
import { HomeIcon, UsersIcon } from "@/assets/icons";
```

---

## Installation & Setup

```bash
# Install lucide-react
npm install lucide-react

# Or with yarn
yarn add lucide-react
```

## Usage Example

```typescript
import { Home, Users, Settings } from "lucide-react";

// Basic usage
<Home />

// With size
<Home size={24} />

// With custom styling
<Home size={24} className="text-blue-500" strokeWidth={2} />

// With color
<Home size={24} color="#3b82f6" />
```
